You may also have to update your server encoding

Make sure that your HTTP header is also UTF-8.

  • If you're using PHP you can use the header() function:
  • header('Content-Type:text/html; charset=UTF-8');
  • There are other ways to make sure your HTTP header is using UTF-8, just depends on your tech.
  • You can find more about this here.

Also in your database

You'll need to set your database storage (tables, columns, etc) and collation to use UTF-8 charset.

PHP gets a bit more complicated, though. You'll have to use multibyte-safe functions to make sure it works.
